# Pabrikin Labs

Welcome to **Pabrikin Labs**, the official website of our industrial solutions company. We specialize in providing cutting-edge manufacturing and energy optimization services for industrial clients.

## About Us

Pabrikin Labs is a strategic partner for manufacturing and industrial energy optimization. We offer comprehensive solutions including system visualization, process optimization, and educational physics simulations.

## Features

### System Visualization
- **Boiler Systems**: Complete stoker boiler system installation and optimization
- **Distribution Networks**: Industrial steam distribution line audits and optimization
- **Distillation Columns**: Crude palm oil distillation process visualization

### Physics Simulations
Interactive educational tools for understanding fundamental physics concepts:
- **GLB (Uniform Linear Motion)**: Position calculation with velocity and time inputs
- **GLBB (Uniformly Accelerated Motion)**: Motion simulation with real-time animation and graphs
- **Mechanical Energy**: Energy conservation principles with interactive diagrams

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Visualization**: SVG, Canvas API, Chart.js
- **Animations**: CSS Animations, JavaScript requestAnimationFrame
- **Responsive Design**: Mobile-first approach

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - runs entirely in the browser

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/pabrikin-labs.git
   ```

2. Navigate to the project directory:
   ```bash
   cd pabrikin-labs
   ```

3. Open `index.html` in your web browser or serve locally:
   ```bash
   # Using Python (if available)
   python -m http.server 8000

   # Or using Node.js
   npx serve .
   ```
